Privacy Policy for JIRA URL Wizard

Last Updated: April 1, 2024

1. Introduction

JIRA URL Wizard is committed to protecting your privacy. This privacy policy explains our data collection, use, and protection practices for the Chrome extension.

2. Information Collection

We Do Not Collect:
  • Personal information
  • Health information
  • Financial information
  • Authentication data
  • Personal communications
  • Location data
  • Web history
  • User activity data
  • Website content

3. Local Storage Usage

The extension stores the following information locally in your browser:

  • Environment configuration preferences
  • URL patterns for different environments
  • UI preferences

Important: This data never leaves your browser and is not transmitted to any external servers.

4. Required Permissions

The extension requires certain permissions to function:

  • activeTab: To read the current tab's URL for transformation
  • storage: To save your environment configurations locally
  • tabs: To manage URL transformations across tabs
  • contextMenus: To provide right-click menu functionality
  • clipboardWrite: To copy generated URLs
  • host: To access and transform JIRA URLs

5. Data Security

We maintain security through:

  • Local-only data storage
  • No external data transmission
  • No third-party service integration
  • No analytics or tracking

6. Your Rights and Control

You have complete control over your data:

  • Clear all data by uninstalling the extension
  • Reset preferences through extension settings
  • No data persistence after removal

7. Changes to Privacy Policy

We will notify users of any privacy policy changes through the extension's update notes and this page.

8. Contact Information

For privacy-related questions, please contact us through:

  • Chrome Web Store developer contact form
  • Extension support page

9. Compliance

This extension complies with:

  • Chrome Web Store Developer Program Policies
  • GDPR principles
  • Browser extension privacy best practices
