1. Introduction

JIRA URL Wizard is committed to protecting your privacy. This privacy policy explains our data collection, use, and protection practices for the Chrome extension.

Our extension is designed with privacy in mind, operating entirely locally within your browser without transmitting any data to external servers.

2. Information Collection

We Do Not Collect:

  • Personal information
  • Health information
  • Financial information
  • Authentication data
  • Personal communications
  • Location data
  • Web history
  • User activity data
  • Website content

Our extension operates on a privacy-first principle, ensuring that no sensitive or personal information is collected or transmitted.

3. Local Storage Usage

The extension stores the following information locally in your browser:

  • Environment configuration preferences
  • URL patterns for different environments
  • UI preferences
  • Recently generated URLs (limited to last 10)
  • Custom environment templates

Important: This data never leaves your browser and is not transmitted to any external servers. All data is stored locally using your browser's storage API.

4. Required Permissions

activeTab

To read the current tab's URL for transformation. Only activated when you explicitly interact with the extension.

storage

To save your environment configurations locally. All data remains in your browser.

tabs

To manage URL transformations across tabs. Used only for core functionality.

contextMenus

To provide right-click menu functionality for quick access to features.

clipboardWrite

To copy generated URLs. Only writes to clipboard, never reads from it.

host

To access and transform JIRA URLs. Limited to URL transformation only.

5. Data Security

We maintain security through:

  • Local-only data storage
  • No external data transmission
  • No third-party service integration
  • No analytics or tracking
  • Regular security updates
  • Transparent permission usage

All data processing occurs locally within your browser, ensuring maximum security and privacy.

6. Your Rights and Control

You have complete control over your data:

  • Clear all data by uninstalling the extension
  • Reset preferences through extension settings
  • No data persistence after removal
  • Export your settings at any time
  • Choose which features to enable

We believe in transparency and user control over their data.

7. Changes to Privacy Policy

We will notify users of any privacy policy changes through:

  • Extension update notes
  • This privacy policy page
  • GitHub repository updates

We are committed to maintaining transparency about any changes to our privacy practices.

8. Contact Information

For privacy-related questions, please contact us through:

GitHub Issues Chrome Web Store

We aim to respond to all privacy-related inquiries within 48 hours.

9. Compliance

This extension complies with:

  • Chrome Web Store Developer Program Policies
  • GDPR principles
  • Browser extension privacy best practices
  • Data protection regulations
  • User privacy rights

We regularly review and update our privacy practices to ensure compliance with the latest regulations and best practices.
